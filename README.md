# JavaScript Array Methods

## Adding & Removing

- `push()` - add to end
- `pop()` - remove from end
- `unshift()` - add to beginning
- `shift()` - remove from beginning

## Transforming

- `map()` - create new array by transforming each item
- `filter()` - create new array with items that pass a condition
- `reduce()` - combine all items into a single value

## Searching

- `find()` - get first item matching a condition
- `findIndex()` - get index of first match
- `includes()` - check if a value exists (returns `true`/`false`)
- `indexOf()` - find the position of a value

## Iterating

- `forEach()` - loop through each item (no new array)

## Others

- `slice()` - copy part of an array (non-destructive)
- `splice()` - add/remove items at a specific index
- `join()` - convert array to string
- `sort()` - sort items
- `reverse()` - reverse order
- `flat()` - flatten nested arrays
- `some()` / `every()` - check if any/all items match a condition

## Suggested Teaching Order for Beginners

1. `push()` / `pop()` / `forEach()` - understand arrays first
2. `map()` / `filter()` - the most used in real projects
3. `find()` / `includes()` / `indexOf()` - searching
4. `reduce()` - after they're comfortable with the basics
5. `slice()` / `splice()` / `sort()` - utility methods last
